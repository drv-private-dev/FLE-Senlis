# Minimal Bootstrap Template

Clean HTML + CSS + JS template.

## Features
- Bootstrap 5 grid
- Light / Dark theme toggle
- Theme persistence (localStorage)
- JSON-driven navigation
- Mobile-first
- No build, no server

## Usage
1. Open `index.html`
2. Edit `data/menu.json`
3. Customize styles in `css/style.css`

Works via file://
